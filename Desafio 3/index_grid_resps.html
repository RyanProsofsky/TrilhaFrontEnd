Grid layout

a) Explique o que é grid layout

O grid é uma forma de estruturação de layouts, ele nos permite trabalhar em duas dimensões sendo linhas e colunas.

b) Cite e dê exemplos das propriedades do Grid Container?

template-columns:   Para colunas usamos o grid-template-columns

grid-template-rows:   Para linhas usamos o grid-template-rows.

grid-template-areas:   O CSS Grid nos permite dar nomes para cada área da nossa grade e depois indicar onde cada elemento deve ir.
Dessa maneira fica simples de entender a estrutura da nossa grid.Fazemos isso com a propriedade grid-template-areas.
Vamos colocar as classes header, sidenav, content e footer nos itens da grid para podermos indicar a posição de cada um deles.

grid-gap:   Nossos itens estão todos colados. Com esse comando podemos indicar qual o espaço entre linhas e colunas.

d) O que é a unidade fr?

É uma um fração do espaço disponivel no container do grid.

e) O que é a notação repeat?

repeat() serve para repetir todas ou uma seção de track listing, então dessa forma repetindo linhas ou colunas.

f) O que são as linhas do Grid?

As linhas no Grid, definem tamanho e quantidade de linhas e colunas.

g) O que é uma célula no Grid?

Uma célula de grid é a menor unidade em um grid. de tabela.

h) O que são áreas do Grid?

Os itens podem se espalhar por uma ou mais células ambas entre linhas ou colunas e isto cria uma área de grade.
